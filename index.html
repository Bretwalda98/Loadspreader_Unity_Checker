Loadspreader Calculator – Help Guide

Version: 2026-02-12 (rev 38.1) • Dark/Light theme • Steel Mat outriggers A/B/C/D • GBP in t/m² and kN/m²

1) Purpose

Checks bending, shear, and bearing stress of timber/steel base mats and estimates Ground Bearing Pressure (GBP). Suitable for outriggers, bogies, and crawler tracks. Results update live; failed checks highlight in red.

2) Units

  • Geometry: m
  • Loads: tonnes (t)
  • Pressures: t/m² and kN/m²
  • Stresses: MPa (N/mm²)

Disclaimer: g = 9.81 m/s² is used for force conversions. 1 t/m² ≈ 9.81 kN/m².

3) How to use

  1. Select Top Contact Type:
    • Crawler Tracks: enter qmax directly.
    • Steel Mat: enter one or more point loads (A/B/C/D) and Wm.
  2. Enter top footprint sizes (Lr, Lw).
  3. Enter base mat geometry (Lsl, Lsb, thickness h, layers).
  4. Enter allowable ground pressure (GBP Allow).

4) Calculation model (workflow)

Steel Mat (per outrigger):
Wtop = P + Wm
qmax = Wtop / (Lr · Lw,eff)
If “Single base mat (overhang)” is enabled and Lw > Lsl, then Lw,eff = min(Lw, Lsl).
Shear reference:
QS = qmax · Lw,eff · Lsb
qB = QS / Lsl
Lb = (Lsl − Lw,eff)/2
Stress checks (as implemented):
Ab = Lsb·1000 · h·1000
Sx = Ab · h·1000 / 6
Mx = 0.5 · qB · Lb2 · g · 106
σB = Mx / (layers · Sx)
τS = (qB · Lb · g · 103) / (layers · Ab)
σC = qmax · g / 103
Ground Bearing Pressure (GBP):
nlen = floor(Lw,eff/Lsl) + 1
nwid = floor(Lr/Lsb) + 1
If “Single base mat (overhang)” is enabled: nlen = nwid = 1.
Abase = (nlen·Lsl) · (nwid·Lsb)
Wbase = ρ · h · layers · Abase
Wtotal = Wtop + Wbase
GBP (t/m²) = Wtotal / Abase
GBP (kN/m²) = GBP (t/m²) · 9.81

6) Eccentricity & stiffness (idea for an advanced mode)

Current results assume uniform pressure over the footprint and perfect contact. Real mats can warp slightly and eccentric loading can increase peak pressure and bending. A validated “advanced mode” could add (ex, ey) eccentricity inputs and a top‑mat stiffness factor to estimate non‑uniform contact.
